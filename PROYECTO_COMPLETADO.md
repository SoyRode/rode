## 💼 Caso de Uso Real

**Escribano necesita calcular ITP+IRPF para una venta:**

1. Abre la app
2. Presiona "IMPUESTOS SOBRE VENTA"
3. Selecciona "URBANO"
4. Ingresa datos (valuación, precio, fechas)
5. Presiona "CALCULAR"
6. App muestra desglose completo
7. Presiona "EXPORTAR" para generar PDF
8. Envía al cliente o imprime
9. Guarda en historial para futuras referencias

**Tiempo total: 2-3 minutos**
